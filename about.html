About KTV LOUNGE

An elite, app-free synchronized karaoke lounge. Elevate your night with seamless karaoke, tailored for the connoisseur.

How It Works

1
Host creates a room
Click "Create Room" on the host device connected to speakers/TV
2
Guests join from their phones
Enter the room code shown on screen — no app download required
3
Search and queue songs
Guests search YouTube and add songs to the queue
4
Sing along!
Karaoke videos play on the main screen with synchronized lyrics. Guests can cheer and react!

Features

✓ App-free guest experience
✓ YouTube karaoke library
✓ Real-time synchronized playback
✓ Guest cheers & reactions
✓ Scoring system
✓ Queue management
✓ Audio relay for remote singers
✓ Echo & effects control
✓ Playlist profiles
✓ Room lock & guest control
